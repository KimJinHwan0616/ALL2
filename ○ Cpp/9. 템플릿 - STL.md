># STL *(Standard Template Library)*
>표준 `템플릿` 라이브러리 ★
>
>### 스마트 포인터, 배열, 사전
```angular2html
#include <memory>    // 스마트 포인터
#include <vector>    // 배열
#include <map>    // 사전

인스턴스.begin();    // 첫 번째 요소
인스턴스.end();    // 첫 번째 요소
```
---

## 배열 *(vector)*
`{값, 값}`
###### <img src = 'img/배열.png' width='100%'>
>선언: `vector<자료형> 이름;`
>```
>vector<int> v;
>```
>
>할당: `이름.reserve(크기);`
>```
>v.reserve(5);
>```
>
>추가: `이름.push_back(값);`
>```
>v.push_back(10);
>v.push_back(11);
>v.push_back(12);
>```
>
>크기
>>확인: `배열_이름.capacity();`, `인스턴스.size();`
>>```
>>v.capacity();    // 5
>>v.size();    // 3
>>```
>
>---
>
>생성+초기화: `vector<자료형> 인스턴스 = {값, 값};`
>```
>vector<int> v = {10, 20};
>```

## 사전 *(map)*
`{키: 값, 키: 값}`
>생성: `map<자료형1, 자료형2> 인스턴스;`
>```
>map<string, int> dict;
>```
>
>추가: `인스턴스[값1] = 값2;`
>```
>dict["수학"] = 100;
>```
