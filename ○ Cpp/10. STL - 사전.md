## Map
`{키: 값, 키: 값}`
```
#include <map>    // 사전

map<std::string, int> dict;    // 생성

dict["수학"] = 100;
dict["영어"] = 70;
dict["국어"] = 50;
```
>생성: `map<키_자료형, 값_자료형> 객체_이름;`
>```
>map<string, int> dict;
>```
>
>추가: `객체_이름[키] = 값;`
>```
>dict["수학"] = 100;
>dict["영어"] = 70;
>dict["국어"] = 50;
>```
>
>삭제 *(요소)*: `객체_이름.erase(키);`
>```
>dict.erase("영어");    // { {"수학":100}, {"국어":50} }
>```
>
>검색 *(요소)*
>>중복X: `객체_이름.find(키);`
>>```
>>dict.find("국어");    //    반복자
>>dict.find("과학");    //    dict.end()
>>```
>>
>>중복O: `객체_이름.equal_range(키);` 
>>```
>>dict.equal_range("수학");    //    {"수학":100}
>>```
