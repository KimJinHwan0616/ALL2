## 네임스페이스
`식별자` 메모리 공간

```
// 선언 //
namespace 네임스페이스
{
    식별자;
     ...
}

// 호출 //
using namespace 네임스페이스;    // import 네임스페이스
using 네임스페이스::식별자;       // from 네임스페이스 import 식별자

네임스페이스::식별자;    // 네임스페이스.식별자
식별자;                // 식별자
```
