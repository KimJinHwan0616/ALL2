## 기본 클래스
`상위`, `부모`
```angular2html
class 기본_클래스
{         
public:
    클래스();     
    ~클래스();    

    virtual 멤버함수;    // 가상함수
      ...
};

private:
    멤버변수;    // 변수_
      ...
};
```

## 파생 클래스
`하위`, `자식` 
```angular2html
class 파생_클래스 : public 기본_클래스
{         
public:
    클래스();    
    virtual ~클래스();    

    멤버함수 override;    // 오버라이딩
      ...
};

private:
    멤버변수;    // 변수_
      ...
};
```
