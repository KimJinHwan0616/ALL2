># 자료형
>
>### 별칭, 변환, 크기
```
unsigned: size_t    
```
---

## 별칭 
>`using 별칭 = 자료형`: 템플릿O
>```
>using PointF = Point<float>;
>``` 
>`typedef 자료형 별칭`: 템플릿X
>```
>typedef T Type;
>```

## 변환 *(자료형1 → 자료형2)*
>`(자료형1)자료형2;`: C
> 
>`자료형1(자료형2);`: C++
```angular2html
(int)7.4;    // 7
(double)10;    // 10.0

int(7.4);    // 7
double(10);    // 10.0 
```

## 크기 *(byte)*
`sizeof(자료형)`
```
sizeof(7.4);    // 8
sizeof(int);    // 4
```
