>### 프롬프트, ubuntu 패키지
---

## 프롬프트 *(prompt)*
`사용자_계정@컴퓨터_이름: 경로 권한`
```
예) jhkim8@ipc:~$, root@ipc:~#
```

>권한
>`$`: 사용자, `#`: root
>
>경로
>`~`: home, `/`: root, `..`: 상위, `.`: 현재

## ubuntu 패키지
`sudo apt-get 명령어` *(apt, Advanced Packaging Tool)*

>`update`: 업데이트
> 
>`upgrade`: 업그레이드
> 
>`install 패키지`: 설치 
> 
>`remove 패키지`: 삭제 *(설정 파일X)*
> 
>`--purge remove 패키지`: 삭제 *(설정 파일O)*
