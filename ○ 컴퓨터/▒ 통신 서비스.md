```angular2html
예) TCP/IP(표준 프로토콜)

p.34 그림
```
>애플리케이션 계층: `통신 서비스` 종류 
>###### <img src = 'img/애플리케이션 계층.png'>
>
>트랜스포트 계층: 전송 `유형` 선택
>```
>TCP: 신뢰성
>UDP: 속도
>
>표 작성하기
>``` 
>
>네트워크: `수신인` 선택
>```
>LAN(Local Area Network)
>WAN(World Area Network)
>인터넷(Internet): LAN + WAN + ... + LAN + WAN 
>```
> 
>데이터 링크 *(Data Link)*: `프레임` → 신호
>```
>노드(Node): 기기(컴퓨터, 라우터, ...)
>통신 매체: 금속선 케이블(구리선), 광섬유 케이블(유리), 무선 
>``` 
> 
>물리 *(통신 매체)*: 비트열 → 신호
>```
>네트워크 장비?(노드 + 통신 매체): NIC(Network Interface Card), 모뎀
>```
