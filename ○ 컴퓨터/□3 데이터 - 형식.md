># 데이터 형식
>
>### XML, JSON, YAML, 
>### proto
---

## XML *(eXtensible Markup Language)*
`HTML`*(가독성↓, 주석O)*
```angular2html
예) 
<module>
    <name>이름</name>
    <location>위치</location>
    <birth>생년월일</birth>
</module>
```

## JSON *(JavaScript Object Notation)*
`자바스크립트 사전`*(가독성↑, 주석X)*
```angular2html
예) {"name": "이름", "location": "위치", "birth": "생년월일"}
```

## YAML *(Yaml Ain't Markup Language)*
`직렬화`*(가독성↑↑)*
```angular2html
예)
---    # 문서 시작

information: 
  - name: "이름"
  - age: 나이
  - job: 
  - "직업1"
  - "직업2"
  - company: false
  - birth: 생년월일
  
...  # 문서 끝
```

## proto *(Protocol Buffers)*
직렬화
```
syntax = "proto2"    // 버전

package = 파일_경로;    // apollo.cyber.proto;
import "외부파일_경로"    // "cyber/proto/qos_profile.proto";

message 이름 {
    optional 자료형 이름 = 번호;
};
```
