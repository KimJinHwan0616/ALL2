## 다층 퍼셉트론 *(MLP, Multi-Layer Perceptron)*
입력층 + `은닉층` + 출력층

+ ### 얕은 신경망 *(Shallow Neural Network)*
  은닉층 `1개`
  ###### <img src='img/얕은 신경망.png'>
  ```
  예) 게이트 - XOR, ...
  ```

+ ### 심층 신경망 *(DNN, Deep Neural Network)* ★
  은닉층 `2개 이상`
